# Installation

Get nvim-luffy up and running on your system in just a few minutes.

## Prerequisites

<Note>
  Make sure you have these installed before proceeding:
</Note>

- **Neovim** 0.9.0 or later (0.10+ recommended)
- **Git** for cloning the repository
- **Node.js** (for LSP servers like TypeScript)
- **Python 3** (for Python LSP and formatters)
- **A Nerd Font** (for icons - recommended: JetBrainsMono Nerd Font)
- **ripgrep** (for Telescope live grep)
- **fd** (optional, for faster file finding)
- **make** (for building Telescope FZF native)

### Installing Neovim

<CodeGroup>
```bash macOS
# Using Homebrew
brew install neovim
```

```bash Ubuntu/Debian
sudo apt install neovim
```

```bash Arch Linux
sudo pacman -S neovim
```

```bash From Source
git clone https://github.com/neovim/neovim
cd neovim
make CMAKE_BUILD_TYPE=RelWithDebInfo
sudo make install
```
</CodeGroup>

## Installation Steps

<Steps>
  <Step title="Backup existing configuration">
    If you have an existing Neovim configuration, back it up first:
    
    ```bash
    # Backup your current config
    mv ~/.config/nvim ~/.config/nvim.backup
    mv ~/.local/share/nvim ~/.local/share/nvim.backup
    mv ~/.local/state/nvim ~/.local/state/nvim.backup
    mv ~/.cache/nvim ~/.cache/nvim.backup
    ```
    
    <Tip>
      You can restore your old config anytime by reversing these commands.
    </Tip>
  </Step>
  
  <Step title="Clone the repository">
    Clone nvim-luffy to your Neovim config directory:
    
    ```bash
    git clone https://github.com/yourusername/nvim-luffy.git ~/.config/nvim
    ```
    
    <Note>
      Replace `yourusername` with the actual repository owner.
    </Note>
  </Step>
  
  <Step title="Install dependencies">
    Install required system dependencies:
    
    <CodeGroup>
    ```bash macOS
    brew install ripgrep fd
    ```
    
    ```bash Ubuntu/Debian
    sudo apt install ripgrep fd-find
    ```
    
    ```bash Arch Linux
    sudo pacman -S ripgrep fd
    ```
    </CodeGroup>
  </Step>
  
  <Step title="Start Neovim">
    Launch Neovim for the first time:
    
    ```bash
    nvim
    ```
    
    The configuration will automatically:
    - Install **lazy.nvim** plugin manager
    - Download and install all plugins
    - Set up the default colorscheme (Bamboo)
    
    <Note>
      The first launch will take a minute or two as plugins are downloaded and compiled. Subsequent launches will be instant.
    </Note>
  </Step>
  
  <Step title="Install LSP servers">
    Once Neovim loads, Mason will automatically install the configured LSP servers. You can check the status:
    
    ```vim
    :Mason
    ```
    
    The following servers will be installed automatically:
    - **ts_ls** - TypeScript/JavaScript
    - **lua_ls** - Lua
    - **pyright** - Python
    - **html**, **cssls** - HTML/CSS
    - **tailwindcss** - Tailwind CSS
    - **eslint** - JavaScript linting
    - **graphql** - GraphQL
    - **prismals** - Prisma
    - **emmet_ls** - Emmet
    
    <Tip>
      You can install additional servers from the Mason UI by pressing `i` on any package.
    </Tip>
  </Step>
  
  <Step title="Verify installation">
    Verify everything is working:
    
    ```vim
    :checkhealth
    ```
    
    This command checks your Neovim installation and reports any issues. Look for:
    - ✓ Neovim version
    - ✓ Python provider
    - ✓ Node.js provider
    - ✓ Clipboard integration
    - ✓ Treesitter parsers
  </Step>
</Steps>

## Directory Structure

After installation, your configuration will look like this:

```
~/.config/nvim/
├── init.lua                    # Entry point
├── lazy-lock.json             # Plugin version lock file
└── lua/
    └── leyland/
        ├── core/
        │   ├── init.lua       # Core module loader
        │   ├── options.lua    # Vim options
        │   ├── keymaps.lua    # Key mappings
        │   └── autocommands.lua # Autocommands
        ├── plugins/
        │   ├── init.lua       # Misc plugins
        │   ├── harpoon.lua    # File navigation
        │   ├── telescope.lua  # Fuzzy finder
        │   ├── nvim-cmp.lua   # Completion
        │   ├── neogit.lua     # Git integration
        │   ├── treesitter.lua # Syntax highlighting
        │   ├── colorscheme.lua # Color themes
        │   └── lsp/
        │       ├── mason.lua  # LSP installer
        │       └── lsp.lua    # LSP config
        ├── lazy.lua           # Plugin manager setup
        └── lsp.lua            # LSP loader
```

## Troubleshooting

<Tip>
  Having issues? Here are some common solutions:
</Tip>

### Plugins not loading

Run `:Lazy sync` to update and install plugins:

```vim
:Lazy sync
```

### LSP not working

Check Mason installation:

```vim
:Mason
```

Reinstall a specific server by pressing `X` then `i` in the Mason UI.

### Telescope not finding files

Make sure `ripgrep` is installed:

```bash
rg --version
```

### Icons not showing

Install a Nerd Font and configure your terminal to use it:

```bash
# macOS
brew tap homebrew/cask-fonts
brew install font-jetbrains-mono-nerd-font
```
