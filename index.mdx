# Welcome to nvim-luffy

**Maybe the one piece was the commits we made along the way.**

nvim-luffy is a modern, feature-rich Neovim configuration built for developers who want a powerful IDE experience without sacrificing speed or simplicity. Named after the legendary pirate, this configuration helps you navigate the seas of code with ease.

## Key Features

<CardGroup cols={2}>
  <Card title="Plugin Management" icon="box">
    Powered by **lazy.nvim** for fast startup times and lazy loading. Plugins are organized and loaded only when needed.
  </Card>
  
  <Card title="LSP Support" icon="code">
    Full Language Server Protocol support with **Mason** for easy LSP installation and management. Pre-configured for TypeScript, Python, Lua, and more.
  </Card>
  
  <Card title="Fuzzy Finding" icon="magnifying-glass">
    Lightning-fast file and text search with **Telescope** and FZF integration. Find anything in your codebase instantly.
  </Card>
  
  <Card title="File Navigation" icon="anchor">
    **Harpoon 2** integration for quick navigation between your most important files. Mark files and jump between them with single keystrokes.
  </Card>
  
  <Card title="Git Integration" icon="git">
    Seamless Git workflow with **Neogit** and **Diffview**. Stage, commit, and review changes without leaving Neovim.
  </Card>
  
  <Card title="Smart Completion" icon="sparkles">
    Intelligent auto-completion with **nvim-cmp**, LuaSnip, and LSP integration. Context-aware suggestions as you type.
  </Card>
  
  <Card title="Syntax Highlighting" icon="palette">
    Advanced syntax highlighting with **Treesitter**. Better code understanding and navigation.
  </Card>
  
  <Card title="Beautiful Themes" icon="brush">
    Multiple colorschemes included: Catppuccin, Tokyo Night, Rose Pine, Bamboo, Modus, and OneDark.
  </Card>
</CardGroup>

## Why nvim-luffy?

<Note>
  This configuration is designed to be both powerful and approachable. Whether you're new to Neovim or a seasoned veteran, nvim-luffy provides a solid foundation that's easy to customize.
</Note>

- **Fast Startup**: Lazy loading ensures Neovim starts in milliseconds
- **Well-Organized**: Clear structure with plugins organized by category
- **Extensible**: Easy to add your own plugins and customizations
- **Modern**: Uses the latest Neovim features and popular plugins
- **Complete**: Everything you need for a productive development environment

## What's Inside

The configuration includes carefully selected plugins that work together seamlessly:

- **Core Editor**: Enhanced options, custom keymaps, and autocommands
- **UI**: Alpha dashboard, custom winbar, minimal statusline
- **Navigation**: Telescope, Harpoon, nvim-tree, Flash for quick jumps
- **Development**: LSP, Mason, auto-completion, Treesitter, formatting
- **Git**: Neogit, Diffview for comprehensive Git workflow
- **Utilities**: Autopairs, Todo comments, Trouble for diagnostics
